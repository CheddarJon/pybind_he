### Install Pybind on Windows Subsystem For Linux (Ubuntu) 

1. Install *pytest* `$ python3.6 -m pip install pytest`

2. Run *install_Pybind.sh*

   ```bash
   $ cd script
   $ chmod u+x install_Pybind.sh
   ```
